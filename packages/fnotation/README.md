# fnotation

[hackage](https://hackage.haskell.org/package/fnotation)

F-notation is a [lower house](https://parentheticallyspeaking.org/articles/bicameral-not-homoiconic/) syntax for programming language experimentation. F-notation is in flux as it is used for new programming languages, so there are some significant differences from the [rust implementation](https://github.com/ToposInstitute/fnotation).

Once I actually release a language which uses this, there will be more documentation. For now, the tests are probably the best way to get an idea of how to use this.
